Octopress Jekyll-Bootstrap 比较

Feb 17th, 2014

less than 1 minute

想为项目建一个项目网站，希望简单快速搭建，又省事好维护，jekyll成为了一个好选择，可惜没主题的网页不好看，好看的主题还要改半天，不符合“简单快速搭建”的原则了。

找到了octopress和jekyll-bootstrap两个工具，拿来试用了下，顺手写个对比。

jekyll-bootstrap

快速开始

通过以下命令就可以开始使用了，用法和jekyll一样

git clone https://github.com/plusjade/jekyll-bootstrap.git
$ cd jekyll-bootstrap
$ jekyll serve

详细文档

优缺点

优点：

  1. 文档详尽（都不需要我再整理什么了）
  2. 用法、语法等和jekyll一致，没有自建一套系统

缺点：

  1. 主题比较少，只有官方的几款主题用起来最省事

octopress

快速开始

// 安装
git clone git://github.com/imathis/octopress.git octopress
cd octopress
gem install bundler
rbenv rehash    # Mac上用不到。If you use rbenv, rehash to be able to run the bundle command
bundle install
rake install    # Install the default Octopress theme
// 创建文章
rake new_post['Post title']
// 生成文章
rake generate
// 发布到github的USERNAME.github.com
rake setup_github_pages
rake deploy
// 安装第三方主题方法
cd octopress
git clone GIT_URL .themes/THEME_NAME
rake install['THEME_NAME']
rake generate
// 本地运行
rake watch // 实时监视本地修改并regenerate
rake preview // 实时监视，并且启动本地webserver，通过http://localhost:4000访问

主题选择

  • 3rd-Party-Octopress-Themes
  • opthemes

优缺点

优点：

  1. 主题丰富

缺点：

  1. 安装麻烦，安装了一些不想加入到系统中的依赖
  2. 自建了一套语法系统，需要再学
